# [912. sort-an-array](https://leetcode.cn/problems/sort-an-array/)

## 题目
**排序数组**

给你一个整数数组 nums，请你将该数组升序排列。


示例 1：

~~~
输入：nums = [5,2,3,1]
输出：[1,2,3,5]
~~~

示例 2：

~~~
输入：nums = [5,1,1,2,0,0]
输出：[0,0,1,1,2,5]
~~~

## 题目大意

排序无序数组

## 解题思路

**方法一：**
1. 使用归并排序
2. 使用快排排序实现
3. 使用冒泡排序实现
4. 使用选择排序实现
5. 使用插入排序实现
