# [543. diameter-of-binary-tree](https://leetcode.cn/problems/diameter-of-binary-tree/)

## 题目
**二叉树的直径**

给定一棵二叉树，你需要计算它的直径长度。一棵二叉树的直径长度是任意两个结点路径长度中的最大值。这条路径可能穿过也可能不穿过根结点。

示例 1：

给定二叉树
~~~
          1
         / \
        2   3
       / \     
      4   5    
~~~

返回 3, 它的长度是路径 [4,2,1,3] 或者 [5,2,1,3]。


## 题目大意

即求解max(左子树)+max(右子树)深度

## 解题思路

**方法一：**
1. 递归遍历
2. 取左子树最大深度+右子树最大深度
